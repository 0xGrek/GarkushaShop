# GarkushaShop

Static shop landing page practice project built with HTML, Sass, JavaScript, Gulp, and BrowserSync.

## Run

```bash
npm install
npm start
```

The source page is `src/index.html`. A root `index.html` redirects to it for simple GitHub Pages usage.

## Scripts

- `npm start` - run the Gulp dev server
- `npm run build:css` - compile and minify Sass

## Notes

- This is a legacy layout practice project.
- It is not an active ecommerce system and does not include cart, payments, backend, or admin functionality.
- `node_modules` should not be committed; dependencies are restored with `npm install`.
